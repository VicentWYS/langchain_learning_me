## 查看现有的包
1. 查看所有已安装的包：打开 `requirements.txt` ,里面包含有本项目安装的全部包，以及对应版本；
2. 查看已安装包的具体版本：在vscode终端中，输入 `pip show python-dotenv`,即可查看版本号；

## 新增包的流程
1. 打开requirements.txt;
2. 添加包：在其中对应位置，添加包名，以及对应版本，如：`python-dotenv>=1.0.0`;
3. 安装包：明确包名称，在vscode终端中，输入 `pip install python-dotenv`;
4. 使用包：在代码中，即可成功引用该包中的方法，如：`from dotenv import load_dotenv`;
